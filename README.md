# Zeekay Store - E-Commerce Application

A complete, production-ready e-commerce web application built with PHP, MySQL, and Monime payment integration.

## Admin Login

Email: admin@zeekay.com
Password: admin123

### User Features

- User registration and login with password hashing
- Product browsing with categories and search
- Shopping cart with quantity management
- Secure checkout with address collection
- Order history and invoice viewing

### Admin Features

- Dashboard with sales statistics
- Product management (CRUD operations)
- Order management with status updates
- User management

### Payment Integration

- Full Monime payment gateway integration
- Automatic order status updates via webhook
- Secure transaction handling
- Success/failure redirects

## Technology Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript, Bootstrap 5
- **Backend:** Core PHP (no framework)
- **Database:** MySQL with PDO
- **Payment:** Monime
- **Security:** Session-based auth, bcrypt hashing, prepared statements

## Installation

### Prerequisites

- PHP 7.4 or higher
- MySQL 5.7 or higher
- XAMPP/WAMP/MAMP or any web server
